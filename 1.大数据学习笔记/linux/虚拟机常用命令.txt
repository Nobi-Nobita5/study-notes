1.vagrant up 启动虚拟机
2.vagrant 其他常用命令
 vagrant ssh：自动使用 vagrant 用户连接虚拟机。
 vagrant upload source [destination] [name|id]：上传文件
 https://www.vagrantup.com/docs/cli/init.html Vagrant 命令行

3.sudo systemctl start docker 启动docker（systemctl 命令是系统服务管理器指令）


docker相关命令：
1.sudo docker images 查看已下载的镜像
2.su root 切换到root用户
3.docker ps 查看正在运行的容器（每个容器就是一个小的linux）
4.docker exec -it mysql mysql -uroot -proot
5.exit 退出容器
6.docker exec -it mysql /bin/bash 进入mysql容器目录


7.docker exec -it java-8 /bin/bash 
-- java-8 ：容器名称
/bin/bash : 进入命令，可使用 docker ps 查看 COMMAND



8.docker update --restart=always zookeeper 修改配置容器自动启动

9.windows 启动Redis
  * 启动redis-server.exe redis.windows.conf
  * 链接客户端redis-cli.exe -h 127.0.0.1 -p 6379
